# gpulse

An LSP for the WebGPU Shader Language (WGSL).

## Goals

- Spec 1:1 Compliance
- Maintain a small binary size
- High performance

## Coverage

gpulse implements the WebGPU Shading Language spec from June 26, 2023.

We currently pass 94 out of 102 of the parser tests.
